# Personal Portfolio Website

My personal website and portfolio designed to showcase my education, career history, technical skills, and open-source contributions.

## Features

* About Me: An overview of my professional background and tech interests.
* Professional Experience: Detailed timeline of my work history and roles.
* Technical Skills: Overview of my expertise in networking, cloud solutions, and frontend technologies.
* Responsive Interface: Clean layout optimized for desktop, tablet, and mobile browsers.

## Built With

* HTML5 and CSS3
* JavaScript
* Tailwind CSS
