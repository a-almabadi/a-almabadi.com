# Networking Basics - For Beginners

## What does "Network" mean?

A Network is a group of connected devices that can share resources and communicate with each other, such as:

- Exchanging data between devices
- Sharing files easily
- Connecting multiple devices to the internet
- Using a single printer from multiple locations

In short, a network simplifies our digital lives and connects all our devices.

## Types of Networks

There are several types of networks, the most important of which are:

- **LAN (Local Area Network)**: Connects devices in one place, like a home or office.
- **WAN (Wide Area Network)**: Connects different locations, like the internet.
- **WLAN (Wireless LAN)**: A wireless network (Wi-Fi) in your home or company.
- **PAN (Personal Area Network)**: A personal network, like connecting your phone to your smartwatch.

## Network Components

Network components are divided into two main categories:

### 1. Intermediary Devices

These manage data traffic and connect devices, such as:

- **Router**: Connects your network to the internet and distributes the connection.
- **Switch**: Connects devices on the network internally.
- **Firewall**: Protects the network from attacks and sets traffic rules.

### 2. End Devices

These are the devices we use directly, such as:

- Phone
- Laptop
- Server
- Printer
- Smart TV

Each device has a way to connect to the network; some are wired (Ethernet), and some are wireless (Wi-Fi).

## Hardware and Software

Any device you interact with consists of:

1. **Hardware**: Such as RAM, hard disk, processor, and network card.
2. **Software**: Such as the operating system (Windows - Linux - macOS), and applications (browser, security programs, network tools). The software gives commands to the hardware to make it work.

## Network Layers (OSI Model) - Detailed Explanation

The OSI (Open Systems Interconnection) model is a universal standard that helps us understand how data is transmitted from one device to another within a network. This model consists of seven layers, each responsible for a specific part of the communication process. Think of it like the stages of delivering a message from one person to another, where each stage does its job and passes it to the next.

The layers from bottom to top:

### 1. Physical Layer

"The foundation where data actually moves." It deals with tangible things like cables, wires, and electrical or light signals. Example: network cable, network card. It transmits 0s and 1s as signals.

### 2. Data Link Layer

"Organizes the connection between two directly connected devices." It deals with the MAC address found in every network card and ensures data arrives without issues between adjacent devices.

### 3. Network Layer

"Chooses the path for the data." It determines the route from source to destination using IP Addresses. It's like Google Maps, but for data. The main protocol here is IP.

### 4. Transport Layer

"Ensures data arrives completely and in order." It uses protocols like TCP (reliable, re-transmits if data is lost) and UDP (fast, but doesn't verify receipt).

### 5. Session Layer

"Starts and ends communication sessions." Example: an online voice call or a login session to a website. It controls when the communication starts and ends.

### 6. Presentation Layer

"Prepares data for display or use." It encrypts and decrypts data (like HTTPS), formats files, and helps the device understand how to read the data.

### 7. Application Layer

"The user interface." This includes the applications we interact with daily, like browsers, email, and FTP. It's the final layer and the closest to us as users.

## Network Addresses (IP Address)

Every device has its own address called an IP Address, like your home address. It helps direct data in and out of the network.

- **IPv4**: The traditional format (e.g., 192.168.1.1).
- **IPv6**: The newer version, which has a huge number of addresses.

## How Do Devices Connect?

Connection is done in two ways:

- **Wired (Ethernet)**: Faster and stable, excellent for high performance.
- **Wireless (Wi-Fi)**: Convenient and easy, but can sometimes be affected by signal or distance.
