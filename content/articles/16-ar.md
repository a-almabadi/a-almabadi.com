# أوامر الأمان في لينكس — Security Commands in Linux

في هذا المقال راح نستعرض مجموعة من أهم أوامر الأمان في نظام لينكس، واللي تساعدك تراقب نشاط جهازك وتحافظ على أمان النظام — سواء تستخدمه كجهاز شخصي، خادم (Server)، أو في بيئة عمل.

## 1. ss 📝 — مراقبة الاتصالات والشبكات

أمر ss من أهم أوامر تحليل الشبكة لأنه يعرض لك كل الاتصالات المفتوحة على جهازك.

**ليش مهم؟**

- يكشف البورتات المفتوحة.
- يوضح الاتصالات النشطة.
- يساعدك تكتشف أي اتصال مشبوه.
- أسرع وأكثر دقة من netstat.

**أمثلة:**

```bash
ss                # يعرض جميع الاتصالات
ss -a             # يعرض جميع البورتات المستمعة وغير المستمعة
ss -t             # يعرض اتصالات TCP فقط
ss -lnt           # عرض المنافذ المستمعة فقط مع التفاصيل
```

## 2. who 👀 — معرفة المستخدمين المتصلين حالياً

يعرض الأمر who جميع المستخدمين اللي مسجلين دخول للنظام في الوقت الحالي.

**ليش مهم؟**

- تعرف إذا فيه أحد داخل على جهازك بدون إذنك.
- مهم بالخوادم اللي يستخدمها أكثر من شخص.

```bash
who
```

## 3. last 💻 — مراجعة سجل تسجيلات الدخول

يعرض الأمر last سجل كل عمليات تسجيل الدخول والخروج.

**ليش مهم؟**

- مراقبة النشاط الأمني للنظام.
- كشف محاولات الدخول غير المعتادة.
- معرفة الوقت والـ IP المستخدم أثناء الدخول.

```bash
last
```

## 4. passwd 🔒 — تغيير كلمة المرور

تغيير كلمات المرور خطوة أساسية لأمان الحسابات.

**نصائح سريعة:**

- استخدم كلمة مرور قوية جدًا.
- غيّرها بشكل دوري.
- عطّل الحسابات غير المستخدمة.

```bash
sudo passwd Abdullah
```

## 5. ufw 🔥 — جدار حماية سهل وفعّال

UFW هو "Uncomplicated Firewall"، أداة بسيطة وسهلة لإدارة جدار الحماية.

**ليش مهم؟**

- يغلق البورتات غير الضرورية.
- يمنع الوصول غير المصرح به.
- ممتاز للخوادم الصغيرة والمتوسطة.

**أمثلة:**

```bash
sudo ufw enable        # تشغيل الجدار الناري
sudo ufw status        # عرض الحالة
sudo ufw allow 22      # السماح بالاتصال عبر بورت معين (مثل SSH)
sudo ufw deny 80       # منع الوصول لبورت معين
sudo ufw delete allow 22
```
