# أوامر مراقبة النظام في لينكس

هذه الأدوات تمكّنك من معرفة ما يحدث داخل جهازك — من استهلاك المعالج والذاكرة إلى نشاط الأقراص والعمليات وحركة الشبكة.

إليك أهم الأوامر:

## 1. uptime

يعرض أمر `uptime` مدة تشغيل النظام منذ آخر إعادة تشغيل.

ويظهر أيضًا:

- الوقت الحالي
- متوسط أحمال النظام (Load Average)
- عدد المستخدمين المتصلين

**مثال:**

```bash
uptime
```

## 2. free

يُستخدم `free` لعرض استخدام الذاكرة.

يعرض:

- إجمالي الذاكرة
- الذاكرة المستخدمة
- الذاكرة المتاحة
- استخدام السواب (Swap)

**مثال:**

```bash
free -h
```

## 3. ps

يعرض أمر `ps` العمليات الحالية التي تعمل في النظام.

**أمثلة:**

```bash
ps
ps aux    # يعرض جميع العمليات
```

## 4. tcpdump

`tcpdump` أداة قوية لتحليل حركة الشبكة Network Traffic.

**مثال:**

```bash
sudo tcpdump
```

## 5. w

يعرض أمر `w` المستخدمين المتصلين حالياً وما يقومون به.

**مثال:**

```bash
w
```

## 6. htop

أمر `htop` عارض تفاعلي للعمليات — يشبه `top` لكنه أكثر تقدماً وسهولة في الاستخدام.

يعرض:

- استهلاك المعالج
- استهلاك الذاكرة
- العمليات في الوقت الحقيقي

**مثال:**

```bash
htop
```

## 7. iostat

أمر `iostat` يُستخدم لمراقبة أداء أجهزة التخزين.

**مفيد في:**

- كشف مشاكل بطء الأقراص
- معرفة سرعات القراءة والكتابة
- مراقبة نشاط الأقراص

**مثال:**

```bash
iostat
```

## 8. lsof

يرمز `lsof` إلى List Open Files.

يعرض:

- الملفات المفتوحة
- العملية التي قامت بفتح كل ملف
- المنافذ المستخدمة من قبل العمليات

**مثال:**

```bash
sudo lsof
```

## 9. pidstat

يُستخدم `pidstat` لمراقبة عملية محددة وتحليل استهلاكها للموارد.

**مفيد في:**

- تتبع ارتفاعات استهلاك المعالج
- معرفة العمليات المتسببة في مشاكل
- تحليل أداء التطبيقات

**مثال:**

```bash
pidstat
```

## 10. top

يعرض أمر `top` أكثر العمليات نشاطًا في الوقت الحقيقي.

يساعدك على معرفة:

- استهلاك CPU
- استهلاك RAM
- متوسط أحمال النظام
- العمليات التي تستهلك موارد عالية

**مثال:**

```bash
top
```
