# إدارة حسابات المستخدمين في لينكس — أوامر لإدارة المستخدمين

مرحبًا جميعًا!

صار فترة طويلة عن آخر مقال لي!

اليوم راجع لكم بمقالة مفيدة عن إدارة حسابات المستخدمين في نظام لينكس.

راح أستعرض أهم الأوامر اللي نستخدمها بشكل يومي. بالنسبة للتوزيعة، اشتغلت على Fedora، لكن الأوامر نفسها تنطبق على أغلب توزيعات لينكس.

## 1. who

أمر `who` يُستخدم لمعرفة المستخدمين المتصلين حاليًا بالنظام.

```bash
a-almabadi@fedora:~$ who

a-almabadi seat0        2023-11-19 10:34
a-almabadi tty2         2023-11-19 10:34
```

## 2. finger

أمر `finger` مشابه لأمر `who`، لكنه يعطي معلومات أكثر تفصيلًا عن كل مستخدم، مثل وقت الدخول والشيل وغيرها.

```bash
a-almabadi@fedora:~$ finger

Login       Name         Tty      Idle  Login Time
a-almabadi  a-almabadi  *tty2     1:54  Nov 19 10:34 2023
```

## 3. useradd

أمر `useradd` يُستخدم لإنشاء حساب مستخدم جديد.

**أفضل استخدام:**

```bash
sudo useradd -m Abdullah
```

الخيار `-m` ينشئ مجلد home للمستخدم تلقائيًا.

بدون هذا الخيار، الحساب يُنشأ بدون مجلد شخصي، وهذا غالبًا غير مرغوب.

هناك تفاصيل أكثر عند إنشاء مستخدم جديد، مثل اختيار الشيل، أو إنشاء الحساب بدون مجلد شخصي. راح أتكلم عن هذا في مقال لاحق بإذن الله.

## 4. passwd

أمر `passwd` يُستخدم لتغيير كلمة مرور المستخدم.

```bash
sudo passwd Abdullah
```

## 5. usermod

أمر `usermod` يُستخدم لتعديل حساب مستخدم موجود.

⚠️ **ملاحظة:** هذا الأمر يحتاج خيارات لتحديد ما تريد تغييره.

**أمثلة:**

```bash
sudo usermod -aG sudo Abdullah      # إضافة المستخدم لمجموعة sudo
sudo usermod -l newname Abdullah    # تغيير اسم المستخدم
sudo usermod -d /new/home Abdullah  # تغيير مجلد home
```

## 6. chfn

أمر `chfn` يُستخدم لتغيير المعلومات الشخصية للمستخدم مثل الاسم الكامل، رقم الهاتف، المكتب، وغيرها.

```bash
sudo chfn a-almabadi
```
