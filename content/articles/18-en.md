# User Account Management in Linux — Commands You Can Use to Manage Users

HEY ALL!

It's been so long!

Today I am back with another informational Linux article! I will be going over a few user management commands in Linux.

The distro I used was [fedora].

## 1. who

The `who` command is used to view who is currently logged in.

```bash
a-almabadi@fedora:~$ who

a-almabadi seat0        2023-11-19 10:34
a-almabadi tty2         2023-11-19 10:34
```

## 2. finger

The `finger` command is similar to `who`, but it provides more details.

```bash
a-almabadi@fedora:~$ finger

Login       Name         Tty      Idle  Login Time
a-almabadi  a-almabadi  *tty2     1:54  Nov 19 10:34 2023
```

## 3. useradd

`useradd` is used to create a new user account.

**Recommended command:**

```bash
sudo useradd -m Abdullah
```

The `-m` option ensures the user's home directory is created.

There is a bit more when it comes to creating a new user, such as creating one with or without a home directory. I will explain more about this in a future article!

## 4. passwd

The `passwd` command is used to change a user's password.

```bash
sudo passwd Abdullah
```

## 5. usermod

`usermod` is used to modify an existing user account.

**Note:** this command requires additional options.

**Examples:**

```bash
sudo usermod -aG sudo Abdullah
sudo usermod -l newname Abdullah
sudo usermod -d /new/home Abdullah
```

## 6. chfn

The `chfn` command is used to change a user's personal information.

```bash
sudo chfn a-almabadi
```
