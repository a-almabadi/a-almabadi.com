# أساسيات الشبكات: شرح موسّع للشبكات الافتراضية VLAN

في هذا المقال سنقدم شرحاً تفصيلياً، من المفاهيم الأساسية إلى التطبيق العملي، مع أمثلة واقعية تستخدمها الشركات.

## ما هي الـ VLAN؟

الـ VLAN هي شبكة محلية افتراضية داخل السويتش.

بدلاً من أن تكون كل الشبكة في Broadcast Domain واحد، تسمح الـ VLAN بتقسيم الشبكة إلى مجالات متعددة دون الحاجة لشراء سويتشات إضافية.

## الهدف الأساسي للـ VLAN

تساعد الـ VLAN على تقسيم الأجهزة داخل الشبكة بناءً على:

- القسم (IT – HR – Finance)
- مستوى الأمان
- نوع البيانات
- الخدمة المطلوبة
- نوع المستخدمين

## لماذا نستخدم VLAN؟

| الفائدة | شرح مبسط |
|---------|----------|
| عزل الترافيك | كل قسم له شبكة خاصة ولا يمكنه الوصول لشبكة أخرى مباشرة |
| رفع الأمان | يقلل من احتمالية sniffing أو الاختراق الداخلي |
| تقليل البث | يقلل حركة البث غير الضرورية ويحسن الأداء |
| تقليل الحمل على الشبكة | تقسيم الشبكة يجعل الترافيك منظم وأكثر كفاءة |
| التحكم في الترافيك | تحديد ما يدخل وما يخرج من كل VLAN |
| سهولة الإدارة والتوسع | إضافة VLAN جديدة أسهل من تركيب سويتش جديد |

## مصطلحات مهمة لفهم الـ VLAN

### 1. Broadcast Domain

جميع الأجهزة التي تستقبل البث من بعضها.

- **بدون VLAN** → الشركة كلها Domain واحد
- **مع VLAN** → كل VLAN يعتبر Domain مستقل

### 2. Access Port

- منفذ مخصص لـ VLAN واحدة فقط
- عادة للأجهزة النهائية (PC، طابعات، إلخ)

### 3. Trunk Port

- منفذ ينقل عدة VLANs بين السويتشات أو بين السويتش والراوتر
- يستخدم 802.1Q tagging لتحديد VLAN الخاص بكل باكيت

### 4. Native VLAN

- VLAN تمر عبر الـ trunk بدون tag
- غالباً تكون VLAN 1، ولكن يُفضل تغييرها لأسباب أمنية

## أنواع VLAN بالتفصيل

### 1. Default VLAN

- غالباً VLAN 1
- كل المنافذ تكون فيها بشكل افتراضي

### 2. Data VLAN

- مخصصة لمستخدمي الشبكة العاديين
- **مثال:**
  - VLAN 10 → المالية
  - VLAN 20 → الموارد البشرية
  - VLAN 30 → التقنية

### 3. Voice VLAN

- مخصصة لهواتف VoIP
- توفر جودة اتصال أعلى وأولوية عبر الشبكة (QoS)

### 4. Management VLAN

- شبكة لإدارة السويتشات باستخدام:
  - SSH، Telnet، SNMP، HTTP/HTTPS
- غالباً VLAN 99 أو VLAN 100

### 5. Native VLAN

- VLAN غير معلمة (untagged) تستخدم بين السويتشات في وضع Trunk

## كيفية إنشاء VLAN على السويتش

### 1. إنشاء VLAN

```
Switch(config)# vlan 10
Switch(config-vlan)# name IT
```

### 2. تعيين VLAN على منفذ Access

```
Switch(config)# interface f0/1
Switch(config-if)# switchport mode access
Switch(config-if)# switchport access vlan 10
```

## الربط بين VLANs (Inter-VLAN Routing)

الأجهزة داخل نفس VLAN تتواصل بشكل طبيعي.

لتواصل VLAN 10 مع VLAN 20، تحتاج إلى:

- راوتر (Router-on-a-Stick)
- أو سويتش Layer 3

### مثال Router-on-a-Stick

```
interface g0/0
 no shutdown

interface g0/0.10
 encapsulation dot1Q 10
 ip address 192.168.10.1 255.255.255.0

interface g0/0.20
 encapsulation dot1Q 20
 ip address 192.168.20.1 255.255.255.0
```

## مثال واقعي من شركات حقيقية

| القسم | VLAN | الشبكة |
|-------|------|--------|
| Finance | 10 | 192.168.10.0 |
| HR | 20 | 192.168.20.0 |
| IT | 30 | 192.168.30.0 |

- كل قسم معزول
- قسم IT يمكنه الوصول لجميع الأقسام
- HR و Finance لا يمكنهم التواصل مباشرة

**خطوات التنفيذ:**

1. إنشاء VLAN لكل قسم
2. استخدام ACL للتحكم بالترافيك
3. تطبيق Inter-VLAN Routing للربط بين الأقسام عند الحاجة

## مشاكل شائعة في VLAN

### VLAN mismatch

- السويتش الأول يرسل VLAN 20 عبر trunk، والسويتش الثاني لا يستقبلها

### Native VLAN مختلفة

- Switch A → Native VLAN 1
- Switch B → Native VLAN 30
- يسبب مشاكل أمنية واتصال

### نسيان إضافة VLAN للـ Trunk

- المنفذ trunk لكن VLAN غير مسموح بمرورها
- **الحل:**
  ```
  switchport trunk allowed vlan 10,20,30
  ```

## أفضل ممارسات أمنية للـ VLAN

- لا تستخدم VLAN 1
- أنشئ Management VLAN خاصة
- أغلق المنافذ غير المستخدمة
  ```
  interface range f0/10-24
   shutdown
  ```
- استخدم Port Security
- اربط VLANs باستخدام ACLs
- أضف فقط VLANs المسموح بها لكل Trunk

## طرق تحسين الشبكة باستخدام VLAN

- عزل أجهزة IoT
- إنشاء Guest VLAN للزوار
- تخصيص VLAN للكاميرات
- تخصيص VLAN للخوادم
- تحسين جودة الصوت والفيديو
- حماية VLAN الإدارة

## الخلاصة

الـ VLAN تقنية أساسية لأي شبكة حديثة.

تتيح لك:

- تنظيم الشبكة
- رفع مستوى الأمان
- التحكم الكامل
- التوسع بسهولة
- أداء عالي

وكل ذلك بدون الحاجة لأجهزة إضافية، باستخدام السويتش نفسه فقط.
